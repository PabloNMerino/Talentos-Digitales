La programación es el proceso de crear instrucciones que una computadora puede seguir para realizar tareas específicas. Utiliza lenguajes de programación, como Python o Java, para escribir código que puede variar desde simples scripts hasta complejos sistemas operativos. Esta disciplina es fundamental en la tecnología moderna, permitiendo el desarrollo de software, aplicaciones y herramientas que facilitan la automatización y la resolución de problemas en diversos campos.

Java es un lenguaje de programación y una plataforma informática que fue desarrollado por Sun Microsystems, ahora propiedad de Oracle Corporation. Fue lanzado en 1995 y es ampliamente conocido por su lema "escribe una vez, ejecuta en cualquier lugar" (WORA, por sus siglas en inglés). Esto se debe a que el código Java, una vez compilado, se convierte en bytecode que puede ejecutarse en cualquier máquina virtual Java (JVM), independientemente del hardware subyacente.
